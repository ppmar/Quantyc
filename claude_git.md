# Git Rules

## Commits
- **No mentions of Claude anywhere** — no Co-Authored-By, no "generated by AI", nothing.
- Commit messages: concise subject line, optional body with bullet points. No attribution lines.

## Pushing
- Before pushing, ensure SSH key is loaded: `ssh-add ~/.ssh/sshkey`
- Push to `origin main` for Railway auto-deploy.

## Security
- **Never commit `.env`, API keys, secrets, or credentials.**
- `.env` must remain in `.gitignore` at all times.
- If a key is accidentally committed, rotate it immediately — git history is permanent.
